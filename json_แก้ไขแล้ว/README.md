# json_แก้ไขแล้ว

โฟลเดอร์นี้เป็น**สำเนาสำหรับตรวจสอบ/แก้ไขความถูกต้อง**ของ raw JSON แต่ละบ้าน (คัดลอกมาจาก `raw_json_ตัวที่ใช้งานจริง/0N<house>/`) — ใช้แก้ไขได้อิสระโดยไม่ต้องขออนุญาตตามกฎข้อ 1 ของ `rule_of_tune.md` (เพราะ path นี้ไม่เข้าข่ายไฟล์ที่กฎคุ้มครอง) แต่ถ้าจะเอาผลที่แก้แล้วไปทับกลับ `raw_json_ตัวที่ใช้งานจริง/` ต้องทำตามกฎข้อ 1-3 เต็มรูปแบบ (เตือนก่อน + log)

## Convention

- ค่าไหนที่มะขามสั่งแก้/ยืนยันเองโดยตรง (ไม่ใช่ Claude เดา) → ตั้ง `confidence_score: 1` เสมอ พร้อม flag อธิบายว่าแก้เพราะอะไร วันที่เท่าไหร่
- ทุกไฟล์ที่แก้ต้อง validate ผ่าน `JSON.parse` ก่อนถือว่าเสร็จ
