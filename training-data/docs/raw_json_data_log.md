## วิธีเพิ่มแถวใหม่

ก่อนแก้ raw JSON ทุกครั้ง (หลังได้รับอนุญาตตามกฎข้อ 1-2 ใน `rule_of_tune.md` แล้ว) ให้เพิ่มแถวใหม่ในตารางด้านบนทันที:

- **วันที่** — วันที่แก้จริง (ISO format `YYYY-MM-DD`)
- **ไฟล์ raw JSON ที่แก้ไข** — path เต็มของไฟล์ เช่น `raw/image/บ้าน_เล็ก_1ชั้น_01/qwen-output/บ้าน_เล็ก_1ชั้น_01_หน้า19.json`
- **AI ที่ใช้ทำ** — ชื่อ/รุ่นโมเดลที่ใช้สกัด/แก้ (เช่น `qwen-vl-max` ของเดิม, `Claude (Sonnet 5, Claude Code)` ถ้าเป็นรอบใหม่)
- **ผู้แก้ไข** — คนที่สั่ง/อนุมัติการแก้ในบทสนทนานั้น (เช่น มะขาม, คิง)
- **หมายเหตุ** — สรุปสั้นๆ ว่าแก้อะไร ทำไม (เช่น "แก้เพราะ Qwen พลาด element F2,C1 หายไปทั้งตัว, ขาด grid แถว D")
