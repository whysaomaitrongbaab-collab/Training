### 1. วินิจฉัยปัญหา "issue loading URL from $image value"
- เช็ค URL ตรงๆ ด้วย `curl` (ทั้งแบบมี/ไม่มี `Origin` header) — พบว่า URL ใช้งานได้ปกติ 100%: `200 OK`, `Content-Type: image/png` ถูกต้อง, และ `Access-Control-Allow-Origin: *` เปิดกว้างอยู่แล้ว (CORS ไม่ใช่ปัญหา)
- สรุปว่าไม่ใช่ปัญหาที่ไฟล์/เซิร์ฟเวอร์ฝั่ง GitHub — น่าจะเป็นปัญหาชั่วคราวฝั่ง browser (rate-limit จาก GitHub CDN ตอนโหลดหลายรูปพร้อมกัน, browser extension บล็อก domain, หรือ cache ค้าง) ให้คำแนะนำลองใหม่ + เช็ค DevTools Network tab
